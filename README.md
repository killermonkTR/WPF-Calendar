# Custom WPF Calendar

This is a university class project that evolved into a unique, fully-featured calendar application. Many Features and upgrades are pending

## Features
* **Modern UI:** Split-pane design, borderless window, and custom colors.
* **Event Management:** Create events, set times, and add notes.
* **Dynamic Filtering:** Searchable dropdowns for contacts and categories.
* **Database Backed:** SQL integration for persistent data storage.
